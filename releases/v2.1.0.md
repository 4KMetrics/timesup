# Time's Up v2.1.0

**Release date:** March 1, 2026

---

## Features

### 1. Auto-Save Configuration

Settings are now saved automatically as you edit them. No more forgetting to
hit save.

- **Debounced 800ms** for text inputs and editable fields — waits until you
  stop typing, then saves
- **Immediate save** for discrete actions (toggles, dropdowns, checkbox changes)
- **Visual indicator** confirms when configuration has been saved

### 2. PIN Protection

Lock down settings and snooze bypass behind a PIN or passphrase.

- Supports **4-digit**, **6-digit**, or **text passphrase** formats
- PIN is **SHA-256 hashed** before storage — the raw value is never persisted
- Gates access to the settings panel and the snooze/bypass flow
- **3-attempt lockout** after consecutive failed entries

### 3. Category Suggestions

Eight pre-built category templates make initial setup faster.

- **Suggestion chips** appear in the category creation UI
- One click populates the category name and a curated list of matching domains
- Templates cover: Social Media, Video, News, Shopping, Gaming, Forums,
  Streaming, and Productivity

### 4. JSON Validation

All configuration fields are now validated on input.

- Type-checks every config field (strings, numbers, arrays, booleans)
- **Inline error display** highlights the specific field that failed validation
- Prevents malformed config from being saved or imported

### 5. "Binge Time?" Button

The "Done for Today" button has been renamed to **"Binge Time?"** with
warning-style visual treatment.

- Orange/red button styling signals the weight of the action
- Functionally identical — disables all blocking for the rest of the day
- Name change makes the consequence more honest and harder to dismiss casually

### 6. Quick-Add Category Buttons

One-click category creation with pre-populated site lists.

- Available for all 8 suggested categories
- Creates the category, populates domains, and sets a reasonable default
  time budget in a single action

---

## Migration Notes

- **Backward-compatible** — no schema changes to stored configuration
- Existing configs are fully compatible; no migration step required
- New fields (PIN hash, auto-save preferences) are added on first use with
  sensible defaults

---

## Known Limitations

- **Import does not validate** — importing a malformed JSON config file may
  produce unexpected behavior
- **PIN lockout resets on page refresh** — the 3-attempt counter is held in
  memory, not persisted
- **Fixed suggestion dictionary** — category suggestions use a static domain
  list and are not updated dynamically
- **No undo for auto-save** — once a change is auto-saved, the previous value
  is not recoverable (use export as a backup)
