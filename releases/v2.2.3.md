# Time's Up v2.2.3

**Release date:** March 12, 2026

---

## Bug Fixes

### 1. Settings no longer disappear after a few minutes

Your blocking rules, time limits, and categories were being lost after 2-3
minutes of inactivity. The extension now reloads your saved settings every time
it wakes up, so your configuration stays intact.

### 2. Quick setup presets are accessible again

If your settings were empty (whether from a fresh install or a reset), the
welcome screen with preset templates (Block Social Media, Focus Mode, Start from
Scratch) now reappears so you can quickly set up your rules again.

---

## What's Changed

- Config is now loaded from storage on every service worker restart, not just on
  install or browser startup
- Options page reads directly from storage as a safety fallback
- Welcome banner re-shows whenever your config is empty
- 8 new automated tests guard against future config persistence issues

---

## Migration Notes

- **No action required** -- this is a drop-in update
- All existing settings and usage data are preserved
- No new permissions needed
