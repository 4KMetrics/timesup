# Release: Time's Up v2.2.0

**Release Date:** 2026-03-02
**Status:** Approved
**Manifest Version:** 3

## Release Summary

v2.2.0 is a tooling release — no user-facing changes. Adds the auto-sync workflow that copies landing page, privacy policy, release notes, and Chrome Web Store screenshots from the private source repo to the public-facing [4KMetrics/timesup](https://github.com/4KMetrics/timesup) site.

## What Changed

- New `sync-public-site.yml` CI workflow publishes whitelisted files on every GitHub Release.
- Baseline set of Chrome Web Store screenshots committed for use in the listing.

## Migration Notes

None — no behavior changes to the extension itself.
