Privacy Policy

Effective: February 2026

Data Collection

None. Time's Up collects no analytics, no telemetry, no tracking data, and serves no ads. No account is required. There is no server. There is no data pipeline. Your browsing data stays on your machine.

Data Storage

All data is stored locally in your browser using chrome.storage.local. Nothing is transmitted over the network. The following data is stored:

  • Configuration — blocking rules, time limits, categories, and preferences
  • Usage data — time spent per site per day, kept for 7 days and automatically deleted
  • Snooze state — temporary bypass records, cleared daily

Data never leaves your browser. It is not synced to any server, cloud service, or third party.

Permissions

Time's Up requests only the permissions required for its functionality. Here is a complete list and why each is needed:

Permission Why
storage Save configuration and usage data locally
alarms Periodic time tracking and daily reset
tabs Track active site for time counting
activeTab Access current tab URL for domain matching
declarativeNetRequest Block sites by redirect
idle Pause tracking when away from the computer
scripting Inject doomscroll detection
host_permissions: all_urls Required for declarativeNetRequest to function on all sites

Third Parties

Time's Up includes no third-party SDKs, libraries, or services. There are zero external network requests.

Contact

For questions about this privacy policy, use the support link on the Chrome Web Store listing.
